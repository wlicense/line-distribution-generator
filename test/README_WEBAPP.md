# 🤖 LINE配信文章生成AIエージェント（Webアプリ版）

AIが自動で17本のLINE配信文を生成するWebアプリです。

---

## ✨ 特徴

- 🌐 **ブラウザで完結**: インストール不要、ブラウザだけで使えます
- 🚀 **17本を自動生成**: 登録直後〜イベント終了後まで17本すべての配信文を生成
- 📊 **スプレッドシート出力**: Google Spreadsheetに完全再現された形式で出力
- 🎨 **Few-shot Learning**: 元データの熱量・センス・トーンを再現
- ⚡ **高速生成**: 1〜2分で17本すべて完成

---

## 🎯 使い方

### 1. Webアプリにアクセス

デプロイ後のURLにアクセスしてください
（例: `https://your-app.streamlit.app`）

### 2. イベント情報を入力

以下の5つの項目を入力します：

| 項目 | 説明 | 例 |
|------|------|-----|
| **コンセプト** | イベント名やキャッチコピー | AI ALL STARS 未来トークセッション2026 |
| **ターゲット** | 想定する参加者層 | AI活用に興味がある起業家・フリーランス |
| **解決する課題** | ターゲットが抱える悩み | AIの導入方法がわからない、時間が足りない |
| **理想の未来** | イベント参加後の変化 | AIを活用して労働時間を減らしながら売上を伸ばす |
| **実績** | 信頼性を示す数字 | 累計40億円の起業家を含む8人が集結 |

### 3. 生成ボタンをクリック

「🚀 17本の配信文を生成する」ボタンをクリック

### 4. 完成！

1〜2分で17本すべての配信文が生成されます。
Google Spreadsheetへのリンクが表示されるので、クリックして確認できます。

## 🎨 出力形式

Google Spreadsheetに以下の内容が出力されます：

### LINE配信
- 配信タイミング
- LINE配信文（300〜500文字）
- 文字数

### メール配信
- 件名（30文字以内）
- プレビュー文（13文字程度）
- 本文（400〜600文字）
- フッター

**デザイン**: 元のスプレッドシートの書式を完全再現
- 色（ダークグレー、ライトグリーン、ライトブルー等）
- セル結合
- テキスト配置
- フォントサイズ
